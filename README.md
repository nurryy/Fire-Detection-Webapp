# 🔥 Fire Detection Web Application

A web-based application for image classification to detect fire using Deep Learning (MobileNetV2). This project was developed during the AI Mastery Program held by Orbit Future Academy (2022).

---

## 📌 Project Overview

This application allows users to upload an image and predicts whether the image contains fire or not using a trained Convolutional Neural Network (CNN).

The system consists of:

- Web interface built with Flask
- CNN model based on MobileNetV2
- Image preprocessing
- Fire / Non-Fire prediction
- Confidence score visualization

---

## 🚀 Features

- Upload image
- Image preprocessing
- Fire detection prediction
- Deep Learning inference
- Simple and responsive web interface
